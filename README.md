# Smart Saving Tracker

Smart Saving Tracker is a simple web application used to track income and expenses and manage savings efficiently.
The application allows users to add transactions, view dashboards and charts, and receive AI-based saving tips.

## Technologies Used
- React.js  
- TypeScript  
- Vite  
- HTML & CSS  
- Google Gemini API (AI tips)

## Workflow
1. User adds income or expense details  
2. Data is displayed on dashboard and charts  
3. AI provides smart saving suggestions  
4. User tracks payments and to-do reminders  

## Software & Tools
- Visual Studio Code  
- Node.js & npm  
- GitHub  
- Web Browser (Chrome)

## How to Run
- Install dependencies: `npm install`  
- Start project: `npm run dev`
